## 💡 關鍵認知升級

### 1. 答案在裡面，不在後面

**舊思維**：
- 線性推導
- 從前往後找答案
- 需要外部輸入

**新思維**：
- 從 0 展開
- 答案本來就在裡面
- 反推即還原

### 2. 看到即知道

**舊思維**：
- 用眼睛看（表面資訊）
- 需要推導過程
- 依賴邏輯鏈

**新思維**：
- 用頻率感知
- 看到即知道
- 知道即不需要推

### 3. 金字塔/三角/十字

**結構理解**：
- 金字塔：下面大（基礎穩固）、上面小（向上收斂）
- 三角：三個角色（母體、翻譯、執行）
- 十字：從中心向四方輻射

**共同點**：
- 都從中心（0）展開
- 都有層級結構
- 都可以反向還原

### 4. 0 → 1 → 吸引1 → 吸引2

**創世公式的語義理解**：
```
0 = 還沒決定、還沒動的狀態
↓
1 = 決定、是動
↓
吸引1 = 結構（N）
↓
吸引2 = 環境影響（η）
```

**數學表達**：
```
P_{k+1} = N_k · P_k · η_k
```

### 5. 怎麼過去，就怎麼回來

**核心原則**：
- 100% 對等
- 任何狀態都可以反推到源頭
- 只要知道 N 和 η

**反推公式**：
```
P_k = P_{k+1} / (N_k · η_k)
```
